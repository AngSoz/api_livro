# Projeto bilioteca

Para rodar o projeto siga as etapas:

### 1. Criar ambiente virtual
```
    python -m venv venv
```

### 2. Ativar o ambiente  virtual
```
    venv/Scripts/activate
```

### 3. Instalar dependencias
```
    pip install -r requirements.txt
```

### 4. Rodar o projeto em modo desenvolmento
```
    fastapi dev main.py
```
